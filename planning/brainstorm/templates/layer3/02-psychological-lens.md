# Template: Psychological Lens

Purpose: Ensure legibility, reduce cognitive load, and align with user perception.

## Inputs
- JTBD flows, MQB charter
- Copy & interaction patterns

## Checks
- Legibility: clarity of intent, language, affordances
- Load: steps, memory requirements, error recoveries
- Feedback: timing, tone, usefulness

## Outputs (Artifacts)
- Legibility Audit: issues + corrections
- Load Budget: max steps/time for golden paths
- Feedback Standards: messages, states, micro-interactions

## Anti-Patterns
- Clever UI over clarity
- Inconsistent feedback; hidden state changes
